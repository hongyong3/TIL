```html

```



##### 2.  다음 태그에서 잘못된 부분을 찾으시오.

```html
href -> src
```



##### 3. 당신은 현재 Resume.html에서 작업중이다. "내 사진"이라는 링크를 누르면 Image폴더 안에 my_photo.png를 띄워주는 a태그 경로를 넣으시오.

```html
<a href="../Image/my_photo.png">내 사진</a>   #상대경로
<a href="/TIL/SSAFY/Image/my_photo.png">내 사진</a>   #절대경로
```
